📊 Wine Quality Analysis & Feature Engineering

📌 Project Overview
This project focuses on exploratory data analysis (EDA) and feature engineering on the Wine Quality dataset. The goal is to uncover key factors affecting wine quality and prepare the data for machine learning models.

🎯 Objectives
Understand the structure and quality of the dataset
Perform data cleaning and preprocessing
Explore relationships between features
Engineer new features for better model performance
Prepare data for machine learning

🗂 Dataset
Source: Wine Quality Dataset
Features include:
Fixed acidity
Volatile acidity
Citric acid
Residual sugar
Chlorides
Alcohol
pH
Sulphates
Quality (target available)

🔍 Data Cleaning
Checked for missing values
Verified data types
Handled inconsistencies

📈 Exploratory Data Analysis (EDA)
Distribution of features
Correlation analysis
Visualization using:
Histograms
Heatmaps
Boxplots

⚙️ Feature Engineering
Feature scaling using StandardScaler
Removal of irrelevant features 
Preparation of dataset for modeling

🛠 Tools & Libraries
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

📊 Key Insights
Alcohol content shows strong influence on wine quality
Some features have weak correlation with quality
Data scaling is important for model performance
